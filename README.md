# AI Article Writer & Summarizer

An AI-powered web application that helps users generate, summarize, and manage articles efficiently. The platform combines modern web technologies with AI capabilities to streamline content creation and analysis.

## Features

- ✍️ AI-powered article generation
- 📄 Intelligent article summarization
- 🔐 User authentication and authorization using JWT
- 👤 User account management
- 📚 Save and manage generated articles
- 🔍 Search and browse articles
- 📱 Responsive and modern UI
- ⚡ Fast performance with Next.js

## Tech Stack

### Frontend
- Next.js
- React.js
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

### AI Integration
- OpenAI/Gemini API (depending on your implementation)
